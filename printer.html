打印机语言

GDI, Graphical Device Interface

        待打印的文件 --计算机--> 光栅格式 --USB--> 打印机

PDL: PCL PS

PCL和PS是两种标准化的页面描述语言(PDL) 

        打印内容 --计算机--> 标准的页面描述文件-打印机控制器--> 光栅图像

PPD, PostScript Printer Description

  • 定制 PostScript 打印机驱动程序的行为
    • 打印机驻留字体
    • 可用介质大小及方向
    • 优化的网频、网角、分辨率
    • 色彩输出
  • PPD 文件是打印机厂商提供的
